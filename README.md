#  The Old Book Exchange & Donation System  
**Major-Assignment-Group-4**

##  Project Overview
The Old Book Exchange & Donation System is a web-based platform that allows users to donate, exchange, request, or sell their used books. This system connects people who want to give away old books with those who need them, promoting affordability, accessibility of knowledge, and environmental sustainability through reuse.

This project is being developed as part of our **Major Assignment** using proper **GitHub collaboration, daily commits, and teamwork**.

---

##  Objectives
- Promote book reuse and donation culture  
- Provide affordable access to educational materials  
- Reduce waste and support recycling  
- Practice real-world software development workflow using GitHub

##  Key Features
- User Registration & Login  
- User Profile Management  
- Upload Old Books for Donation or Sale  
- Request Books from Other Users  
- Exchange Books Between Users  
- Admin Panel for Approvals and Monitoring  
- Book Availability Status  
- Search & Filter Books  
- User Notifications  

---

##  User Roles
- **Users:** Donate, sell, request, and exchange books  
- **Admin:** Approve books, manage users, and monitor activities
